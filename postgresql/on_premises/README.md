# PostgreSQL On-Premises Distributions

This directory contains learning materials for different PostgreSQL distributions and variants that run on-premises.

## Distributions

- **open_source** - Standard PostgreSQL (Community Edition)
  - Official PostgreSQL from postgresql.org
  - Core PostgreSQL features and capabilities
  
- **edb** - EnterpriseDB Postgres Advanced Server
  - Enterprise-grade PostgreSQL with additional features
  - Oracle compatibility features
  - Advanced security and management tools
  
- **citus** - Citus (PostgreSQL extension for distributed databases)
  - Horizontal scaling for PostgreSQL
  - Distributed tables and queries
  - Multi-tenant SaaS applications
  
- **greenplum** - Greenplum Database
  - Massively parallel processing (MPP) database
  - Analytics and data warehousing
  - Based on PostgreSQL
  
- **timescaledb** - TimescaleDB
  - PostgreSQL extension for time-series data
  - Automatic partitioning by time
  - Optimized for IoT, monitoring, and analytics
  
- **postgis** - PostGIS (Spatial and Geographic Objects)
  - PostgreSQL extension for geographic information systems
  - Spatial data types and functions
  - Geographic queries and analysis

## Learning Path

Each distribution directory can contain:
- Installation and setup
- Configuration and tuning
- Feature-specific guides
- Use cases and best practices
- Migration from standard PostgreSQL
- Performance optimization
