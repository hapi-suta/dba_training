# PostgreSQL Cloud Platforms

This directory contains learning materials for PostgreSQL running on different cloud platforms.

## Cloud Providers

- **AWS** - Amazon RDS for PostgreSQL / Aurora PostgreSQL
  - Managed PostgreSQL service on AWS
  - Aurora PostgreSQL (PostgreSQL-compatible)
  
- **GCP** - Google Cloud SQL for PostgreSQL
  - Managed PostgreSQL service on Google Cloud Platform
  - Cloud SQL instances and configurations
  
- **Azure** - Azure Database for PostgreSQL
  - Managed PostgreSQL service on Microsoft Azure
  - Flexible Server and Single Server options

## Learning Path

Each platform directory can contain:
- Setup and configuration guides
- Migration strategies
- Performance optimization
- Backup and disaster recovery
- Security best practices
- Cost optimization
- Platform-specific features
